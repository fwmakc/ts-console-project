# Начало работы

Создаем новую папку для проекта

```
mkdir my_project
cd my_project
```

Клонируем репозиторий

```
git clone https://github.com/fwmakc/ts-console-project.git .
```

Устанавливаем

```
yarn
```

# Быстрый запуск

В режиме разработки

```
yarn dev
```

# Запуск в продакшн

Сборка

```
yarn build
```

и запуск

```
yarn start
```
